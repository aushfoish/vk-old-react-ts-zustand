# Vkontakte.ru React

Архивная первая версия точной копии интерфейса ВКонтакте начала-середины 2010-х годов с ограниченным функционалом. Перенос аналогичного проекта (https://github.com/aushfoish/vk-old-design-js) с Vanilla-JS на React-TS 

## Цель проекта

* Изучение возможностей работы с современными фреймворками и сетевыми запросами

## Реализовано:

* Кастомный MP3-плеер с загрузкой данных из облака. В отличии от первой версии добавлена возможность перемотки через полоску хронометража, динамическое обновление таймеров и загрузка 
* Маршрутизация через React Router Dom
* Возможность фонового воспроизведения музыки при навигации по страницам сайта


## В разработке:

* Полноценная пересобранная рисовалка граффити на Canvas API
* Авторизация
* Возможность отправлять посты на стену через REST API
* Различные QoL - улучшения



### Технологии
* React
* Typescript
* Zustand
* React Router DOM
